## 25. K 个一组翻转链表|链表|困难|模拟

### 题目描述

```
给你链表的头节点 head ，每 k 个节点一组进行翻转，请你返回修改后的链表。

k 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 k 的整数倍，那么请将最后剩余的节点保持原有顺序。

你不能只是单纯的改变节点内部的值，而是需要实际进行节点交换。
```

### 解题思路

回顾一下 反转链表 Ⅱ 那道题,将链表中的一个区间反转

其实我们这里就是反转多个长度为 k 的区间罢了

具体多少个区间，就是链表总长度和 k 的关系了

### 代码

```js
/**
 * @param {ListNode} head
 * @param {number} k
 * @return {ListNode}
 */
var reverseKGroup = function (head, k) {
	const len = getListLength(head);
	let count = Math.floor(len / k);
	let hair = new ListNode();
	hair.next = head;
	let pre = hair,
		cur = pre.next;
	while (count--) {
		reverse(pre, cur, k);
		// 每反转一组，要更新 pre 和 cur
		pre = cur;
		cur = cur.next;
	}
	return hair.next;
};
const getListLength = head => {
	let cur = head;
	let i = 0;
	while (cur) {
		cur = cur.next;
		i++;
	}
	return i;
};

const reverse = (pre, cur, k) => {
	//k个点，反转 k-1 次相邻两个节点
	while (--k) {
		// 反转相邻两个节点
		let nxt = cur.next;
		cur.next = nxt.next;
		nxt.next = pre.next;
		pre.next = nxt;
	}
};
```
